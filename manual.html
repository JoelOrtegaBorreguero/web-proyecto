Soporte | Manual

iOS

Acerca el chip NFC a la parte posterior de tu dispositivo iOS. Aparecerá una notificación en tu dispositivo indicando que se ha detectado una etiqueta NFC.

Toca la notificación para interactuar con el chip NFC. La acción que se realiza al tocar la notificación depende del chip NFC. Por ejemplo, si el chip NFC es una tarjeta de transporte, se abrirá una aplicación para que puedas pagar tu viaje.

Android

Activa NFC en tu dispositivo Android. La forma de hacerlo varía según el dispositivo. Consulta el manual de usuario de tu dispositivo para obtener instrucciones específicas.

Acerca el chip NFC a la parte posterior de tu dispositivo Android. Aparecerá una notificación en tu dispositivo indicando que se ha detectado una etiqueta NFC.

Toca la notificación para interactuar con el chip NFC. La acción que se realiza al tocar la notificación depende del chip NFC. Por ejemplo, si el chip NFC es una tarjeta de transporte, se abrirá una aplicación para que puedas pagar tu viaje.
